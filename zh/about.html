我们的使命

在日常的开发工作和内容创作中，我们经常需要处理各种格式的文本数据——格式化 JSON、提取链接、生成二维码等等。虽然市面上有很多类似的工具，但我们发现它们往往存在一些共同的问题：

  • 界面过时、操作不够直观
  • 需要注册登录才能使用
  • 功能分散在不同的网站上
  • 将用户数据上传到服务器，存在隐私风险
  • 广告过多、严重影响使用体验

因此，我们决定创建文本处理工具集——一个将常用的文本处理工具整合在一起的平台，提供优雅的界面、流畅的交互，同时保证所有数据处理都在用户的浏览器本地完成。

我们的工具

🔍

JSON 可视化工具

强大的 JSON 格式化、验证和可视化工具。支持语法高亮、层级颜色编码、树形结构展示、路径复制等高级功能，是开发者处理 JSON 数据的得力助手。

📎

富文本链接提取器

从富文本中智能提取超链接并转换为纯文本格式。保留文本加粗格式和换行结构，支持批量处理，一键复制转换结果。

📱

二维码生成器

将任意文本内容快速转换为二维码图片。支持自定义尺寸和容错级别，即时预览生成结果，方便下载使用。

🕐

时间戳转换

Unix 时间戳与可读日期互转。自动识别秒级和毫秒级时间戳，支持多格式输出（本地时间、UTC、ISO）。

🔗

URL 编解码

URL 文本的编码与解码转换。支持 encodeURIComponent/encodeURI 两种模式，满足不同场景需求。

🔤

Base64 编解码

Base64 编码与解码转换，完整支持中文和 Unicode 字符。编码传输、数据嵌入等场景必备工具。

#️⃣

哈希生成

支持 MD5、SHA-1、SHA-256、SHA-512 四种哈希算法。使用浏览器原生 Web Crypto API 计算哈希值，快速准确。
