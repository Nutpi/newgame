# 🛠️ 多功能文本处理工具集

一个功能强大的在线文本处理工具集，提供富文本链接提取、JSON 数据可视化和二维码生成功能。

## 🚀 功能特性

### 📎 富文本超链接提取转换工具
- **智能链接提取**：从富文本中提取超链接并转换为纯文本格式
- **格式保持**：保留文本的加粗格式和换行结构
- **一键复制**：转换结果可一键复制到剪贴板
- **实时预览**：即时显示转换结果
- **批量处理**：支持多个链接同时处理

### 🔍 JSON 可视化工具
- **语法高亮**：彩色显示 JSON 数据结构
- **树形展示**：层级化展示 JSON 数据
- **展开折叠**：支持节点的展开和折叠操作，带有优美的动画效果
- **类型指示**：清晰标识数据类型（字符串、数字、布尔值等）
- **格式化功能**：美化和压缩 JSON 数据
- **数据验证**：检测 JSON 格式错误并提供详细错误信息
- **结构分析**：显示数据统计信息（属性数量、数组长度等）
- **属性计数高亮**：直观显示对象属性数量

### 📱 二维码生成器
- **文本转二维码**：将任意文本转换为二维码
- **高清输出**：支持高分辨率二维码生成
- **即时预览**：实时显示生成的二维码
- **下载功能**：支持下载二维码图片

## 🎨 界面特性

- **多主题支持**：
  - 🌞 浅色模式：经典白色主题
  - 🌙 深色模式：护眼深色主题
  - 💙 蓝色科技：科技感蓝色主题
  - 💚 绿色护眼：舒适绿色主题
  - 💜 紫色优雅：优雅紫色主题
- **响应式设计**：完美适配桌面端和移动端
  - 桌面端：左右布局，提高工作效率
  - 移动端：上下布局，优化触屏体验
- **现代化 UI**：简洁美观的用户界面
- **流畅动画**：丰富的交互动效和过渡效果
- **无障碍设计**：支持键盘导航和屏幕阅读器

## 📱 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **样式**：CSS3 + CSS Variables + Flexbox + Grid
- **响应式**：Media Queries
- **动画**：CSS Transitions & Transforms
- **部署**：静态网站托管（支持 Cloudflare Pages、GitHub Pages、Vercel 等）

## 🚀 快速开始

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/你的用户名/newgame.git
cd newgame
```

2. 启动本地服务器
```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 PHP
php -S localhost:8000
```

3. 打开浏览器访问 `http://localhost:8000`

### 在线部署

#### GitHub Pages 部署
1. Fork 本仓库到你的 GitHub 账户
2. 进入仓库设置页面
3. 找到 Pages 选项
4. 选择源分支（通常是 `main` 或 `master`）
5. 保存设置，等待部署完成
6. 访问 `https://你的用户名.github.io/newgame`

#### Cloudflare Pages 部署
1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 进入 Pages 服务
3. 连接 GitHub 仓库
4. 选择 `newgame` 仓库
5. 设置构建配置：
   - 构建命令：留空
   - 构建输出目录：`/`
6. 点击部署

#### Vercel 部署
1. 访问 [Vercel](https://vercel.com/)
2. 导入 GitHub 仓库
3. 保持默认设置
4. 点击部署

## 📖 使用说明

### 富文本链接提取器
1. 在输入区粘贴包含超链接的富文本内容
2. 点击「转换」按钮
3. 在输出区查看转换结果
4. 点击「复制」按钮将结果复制到剪贴板
5. 点击「清空」重置输入内容

### JSON 可视化工具
1. 在输入框中粘贴或输入 JSON 数据
2. 点击「格式化」查看美化后的 JSON
3. 点击「压缩」获取压缩版本
4. 点击「验证」检查 JSON 格式
5. 在可视化区域查看树形结构
6. 点击节点前的箭头展开/折叠内容
7. 查看属性数量和数据类型高亮显示

### 二维码生成器
1. 在文本框中输入要转换的内容
2. 点击「生成二维码」按钮
3. 查看生成的二维码预览
4. 点击「下载」保存二维码图片

## 🔧 自定义配置

### 修改主题色彩
编辑 `css/style.css` 文件中的 CSS 变量：

```css

  --accent-primary: #4285f4;    /* 主色调 */
  --accent-secondary: #34a853;  /* 辅助色 */
  --accent-warning: #ff9800;    /* 警告色 */
  --accent-error: #f44336;      /* 错误色 */
}
```

```javascript
class NewTool {

    this.init();
  }
  
  init() {
    // 初始化逻辑
  }
}
```
